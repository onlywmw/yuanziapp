# 图谱参数面板

> **核心**: 图谱的每一个视觉维度都是可调的。混音台不只是管"显示什么", 还管"怎么显示"。
> **灵感**: 调音台 — 每个推子控制一个参数, 一键预设切换风格

## 二、每个参数的作用

### 节点大小

```
控制: 所有节点的基础缩放倍率
范围: 0.5x ~ 2.0x

  0.5x: 节点缩小一半, 适合密集图谱
  1.0x: 默认大小
  2.0x: 节点放大, 适合展示少数重点原子

和混音台的关系:
  管道视角: 基础原子和基础设施节点保持原大, 终端原子缩小
  作品视角: 终端原子放大, 管道节点缩小
```

### 文字透明度

```
控制: 节点标签的不透明度
范围: 0% ~ 100%

  0%:   纯图形, 无文字 (极简)
  50%:  文字半透明, 不影响图形
  100%: 文字完全不透明, 清晰可读

和混音台的关系:
  管道视角: 基础原子标签 100%, 终端原子标签降低
  作品视角: 终端原子标签 100%, 管道标签降低或隐藏
```

### 连线粗细

```
控制: 所有连线的描边宽度
范围: 0.5dp ~ 5dp

  0.5dp: 极细线, 几乎看不见 (适合只看节点)
  2dp:   默认
  5dp:   粗线, 血管感强烈 (适合看数据流)

和混音台的关系:
  管道视角: 连线粗, 通道类型明显
  作品视角: 连线细, 不抢节点视觉
```

### 连线长度

```
控制: 相连节点间的目标距离 (力导向 restLength)
范围: 80dp ~ 300dp

  80dp:  节点挤在一起, 紧凑
  180dp: 默认
  300dp: 节点拉开, 连线长, 适合看整体拓扑

和混音台的关系:
  管道视角: 连线短, 节点紧凑 (看依赖关系)
  作品视角: 连线长, 节点舒展 (看每个作品)
```

### 向心力

```
控制: 所有节点向 CENTER 节点聚拢的力度
范围: 0.0 ~ 1.0

  0.0: 无向心力, 节点自由扩散
  0.5: 默认, 节点围绕 CENTER 分布
  1.0: 最大向心力, 所有节点紧贴 CENTER

和混音台的关系:
  管道视角: 向心力强 (中心是系统核心)
  作品视角: 向心力弱 (每个作品都是自己的中心)
```

### 节点排斥力

```
控制: 所有节点互相排斥的力度
范围: 0.0 ~ 1.0

  0.0: 无排斥, 节点可以重叠
  0.5: 默认, 节点散布均匀
  1.0: 最大排斥, 节点飞散到边缘

和混音台的关系:
  管道视角: 排斥弱 (紧凑, 看关系)
  作品视角: 排斥强 (舒展, 看个体)
```

### 相连吸引力

```
控制: 有连线关系的节点互相吸引的力度
范围: 0.0 ~ 1.0

  0.0: 无额外吸引, 连线只是装饰
  0.5: 默认, 相连节点靠近
  1.0: 最大吸引, 相连节点紧贴

和混音台的关系:
  管道视角: 吸引力强 (依赖关系明显)
  作品视角: 吸引力弱 (节点独立)
```

### 布局速度

```
控制: 力导向算法的收敛速度
范围: 慢 ~ 快

  慢: 节点缓慢移动, 像在糖浆里 (柔和)
  快: 节点快速移动, 像在水里 (响应快)

和混音台的关系:
  管道视角: 快 (调试需要即时反馈)
  作品视角: 慢 (浏览需要美感)
```

## 三、配色方案 — 一键切换

### 按路径配色

```
按 atom_id 的命名空间着色:
  system.*    → 灰色
  mcp.*       → 按域名分段着色
  com.*       → 按组织着色

效果: 看到原子的"来源地图"
```

### 按标签配色

```
按 classification.tags 着色:
  database     → amber
  cloud        → clay_light
  security     → rust
  ai           → rust_deep
  document     → sage

效果: 看到原子的"功能地图" ← 当前默认
```

### 按关键词配色

```
按 soul.style 或 soul.audience 着色:
  minimalist   → 白
  reliable     → 蓝
  creative     → 紫
  backend      → 绿

效果: 看到原子的"风格地图"
```

### 按属性配色

```
按任意 JSON 路径的值着色:
  lifecycle.status   → running=绿, offline=红, registered=蓝
  quality            → polished=金, experimental=灰
  ratings.average    → 渐变 (低=红, 高=绿)

效果: 看到原子的"状态地图"
```

### 配色预设切换

```
切换时:
  节点颜色渐变过渡 (500ms)
  连线颜色同时变化
  颜色图例显示在右上角
```

## 四、混音台作为主控

```
管道 ◄══════════●══════════► 作品

滑块不只是调节可见度, 它是一组参数的快捷控制:

滑块=0.0 (纯管道):
  节点大小      1.0x
  文字透明度    100%
  连线粗细      4dp
  连线长度      120dp (短)
  向心力        0.8 (强)
  排斥力        0.3 (弱)
  吸引力        0.8 (强)
  配色方案      按标签
  布局速度      快

滑块=0.5 (默认):
  节点大小      1.0x
  文字透明度    70%
  连线粗细      2dp
  连线长度      180dp
  向心力        0.5
  排斥力        0.5
  吸引力        0.5
  配色方案      按标签
  布局速度      中

滑块=1.0 (纯作品):
  节点大小      1.4x (大)
  文字透明度    100%
  连线粗细      1dp (细)
  连线长度      250dp (长)
  向心力        0.2 (弱)
  排斥力        0.8 (强)
  吸引力        0.2 (弱)
  配色方案      按soul风格
  布局速度      慢
```

**主控调完后, 每个参数仍可单独微调。**

## 五、预设管理

```
内置预设:
  [🔧 调试]    管道=1.0  紧凑/快速/标签配色
  [⚡ 工作]    管道=0.7  适中/中速/标签配色
  [🎨 浏览]    管道=0.3  舒展/慢速/soul配色
  [💎 发现]    管道=0.0  舒展/慢速/soul配色
  [🎛 默认]    管道=0.5  均衡/中速/标签配色

自定义预设:
  调好参数 → [💾 保存预设] → 命名 → 出现在预设列表
  最多保存 5 个自定义预设
```

## 六、存储

```
所有参数保存在本地 SharedPreferences:

  graph.node_size         = 1.0
  graph.text_opacity      = 0.7
  graph.edge_thickness    = 2.0
  graph.edge_length       = 180
  graph.centripetal       = 0.5
  graph.repulsion         = 0.5
  graph.attraction        = 0.5
  graph.layout_speed      = 0.5
  graph.color_scheme      = "tag"
  graph.mixer_position    = 0.5

启动时恢复上次的参数。
```
