# Yuanzi 前端 UI 设计规格

> **状态**: `📐 design-ready`
> **目标**: Android 知识图谱 + 市场 + 工作流画布的视觉标准
> **设计原则**: 温润、克制、信息优先、动效为功能服务

---

## 一、设计语言

### 色彩系统

```
主色 (clay 陶土):
  clay           #B85C38    主按钮、选中态
  clay_light     #D4956E    浅色背景
  clay_deep      #8B3A1E    深色强调

功能色:
  sage           #6B8E6B    成功、在线
  sage_light     #A3C4A3    已连接
  sage_deep      #4A6B4A    通过审查
  rust           #C23B22    错误、离线、危险
  rust_light     #E8A090    警告背景
  rust_deep      #8B1A0A    严重安全
  amber          #D4A017    警告、未连接
  amber_light    #F0D080    提示背景

中性色:
  ink            #2C2C2C    主文字
  muted          #8C8C8C    辅助文字、section 标题
  faint          #C0C0C0    占位文字
  hairline       #E0E0E0    分割线
  paper          #FAFAFA    页面背景
```

### 字体

```
标题:   22sp, Typeface.BOLD, ink
正文:   15sp, Typeface.NORMAL, ink
辅助:   13sp, Typeface.BOLD, muted
说明:   12sp, Typeface.NORMAL, muted

```

### 间距

```
页面边距:   20dp
卡片内边距: 16dp

```

### 2.2 节点样式

```
CENTER (组件MCP):
  圆形, 56dp 半径
  底色: clay, 文字: paper
  标签: "组件\nMCP"
  固定位置, 不可拖动

YUANZI (中枢):
  圆形, 48dp 半径
  底色: sage(已连接) / rust(断开) / amber(未配置)
  文字: paper
  标签: "Yuanzi\n中枢"
  点击 → 跳转设置页

BROWSER (浏览器):
  圆形, 44dp 半径
  底色: clay_deep, 文字: paper
  标签: "浏览器"
  点击 → 打开浏览器

BASE (基础原子):
  圆角矩形, 40dp 半径
  底色: #F0F0F0, 边框: 灰色虚线 1dp
  文字: muted, 字号: 12sp
  标签: 原子名 (如 "file-read")
  不可拖动, 不可删除, 不可点击

REGISTERED (注册原子):
  圆角矩形, 42dp 半径
  底色: 按分类着色, 边框: 白色实线 1dp
  文字: ink, 字号: 13sp
  标签: 原子名 (如 "mcp.postgres")
  点击 → 原子详情浮层

AUTHOR (作者节点):
  菱形, 36dp 半径
  底色: amber, 文字: paper
  标签: 作者名
  点击 → 显示该作者所有原子
```

### 2.3 连线样式

```
CENTER → 功能节点:
  实线, 2dp, hairline (#E0E0E0)
  无动画

功能节点 → 功能节点:
  直通线: 实线 1.5dp, 箭头
  映射线: 虚线 1.5dp, 标注字段名
  转换线: 波浪线 1.5dp, 标注类型名

数据流动画 (可选开启):
  红色圆点沿输出线流动 (2px/帧)
  蓝色圆点沿输入线流动 (1px/帧)
```

### 2.4 交互

```
单点: 选中节点, 高亮边框
长按: 拖拽移动 (CENTER 和 BASE 除外)
双击: 原子详情浮层
双指: 缩放画布 (0.5x ~ 3x)
单指滑动: 平移画布
```

## 五、工作流画布

### 5.1 布局

```
┌──────────────────────────────────────┐
│  ← 返回   工作流名称   [▶ 运行]     │  ← 顶部工具栏
├──────────────────────────────────────┤
│                                      │
│  ┌──────────────────────────────┐    │
│  │         工作流画布            │    │
│  │                              │    │
│  │  [http-get] ═══[映射]═══    │    │
│  │      │              │       │    │
│  │      │         [json-parse] │    │
│  │      │              │       │    │
│  │  [date-time]──[合并]──→    │    │
│  │                     │       │    │
│  │                [file-write] │    │
│  │                              │    │
│  └──────────────────────────────┘    │
│                                      │
├──────────────────────────────────────┤
│  [＋添加节点]  [＋添加连线]          │  ← 底部操作栏
│  基础原子 ▼   注册原子 ▼            │  ← 展开菜单
└──────────────────────────────────────┘
```

### 5.2 拖拽连线

```
从节点输出端 (右侧圆点) 拖出
  → 显示跟随手指的虚线
  → 松手在目标节点输入端 (左侧圆点)
  → 弹出连线类型选择: 直通 | 映射 | 转换 | 合并 | 分流
  → 选中后连线上出现类型标注
  → 连线颜色按数据类型自动变化
```

### 5.3 运行中动画

```
待执行节点: 灰色边框
执行中节点: amber 边框 + 脉冲动画
成功节点:   sage 边框 + 对勾图标
失败节点:   rust 边框 + 红色闪烁
运行中连线: 数据流动画 (小圆点沿连线移动)
```

## 六、动画规范

```
节点出现:
  新原子注册后, 在知识图谱中从中心扩散出现
  动画: scale 0 → 1, alpha 0 → 1, 持续 300ms, 缓出

连线绘制:
  工作流新增连线时, 从源节点向目标节点生长
  动画: path 从 0% 延伸到 100%, 持续 200ms

数据流动:
  运行中的连线上, 小圆点平滑移动
  速度: 2px/帧 (60fps), 循环

状态切换:
  节点颜色变化 (如 running→unreachable)
  动画: 颜色渐变 500ms

页面切换:
  ViewFlipper 使用淡入淡出
  持续 200ms
```

## 七、响应式

```
手机 (< 600dp 宽):
  知识图谱: 单列, 节点缩小 20%
  市场: 单列卡片, 全宽
  工作流: 单列画布

平板 (≥ 600dp 宽):
  知识图谱: 全尺寸节点
  市场: 双列卡片
  工作流: 全宽画布, 侧边栏显示工具箱
```

---

> **视觉语言: 陶土温润、绿色表示连接、灰色表示不可操作、动效只在有数据流动时出现。**
