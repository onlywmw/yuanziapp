# 共振 — 以人为本

> **核心**: 不是原子和原子共振。是原子和人此刻的需要共振。
> **区别**: 系统不计算"什么和什么匹配"。系统理解"这个人现在需要什么"。

---

## 一、先有人, 再有共振

```
冷的计算:
  weather: {雨强:0.8}  ×  music: {慢:0.7}  →  共振 0.56
  原子对原子。人是接收者。

暖的理解:
  此刻的你:
    刚从雨里走进来 → 身体有点冷 → 需要暖
    坐下 5 分钟 → 还没完全放松 → 需要过渡
    明天没有安排 → 不需要赶 → 需要慢

  原子们在问:
    "我能回应你的哪个需要？"
    
    weather: "我带来了雨声——你需要安静而不是热闹"
    music:   "我可以很慢——你需要过渡而不是直接进入"
    coffee:  "我在你手边——你需要暖"
    device:  "我隔离了外面的噪音——你需要不被干扰"
    calendar:"你还有很久——不需要赶"
```

## 二、人的状态, 不是原子的维度

```
不是 atom.field。是 person.moment。

  身体:    冷→暖过渡中, 从雨里进来 5 分钟
  心理:    想要安静, 不想被催促, 不想做决定
  节奏:    慢, 不赶, 明天休息
  环境:    雨声, 咖啡, 靠窗, 没人打扰
  时间:    刚开始, 不是快结束了, 还有很久

人的状态是流动的。
不是"下雨+咖啡厅=忧郁"。
是"刚到+冷+安静+不赶时间=需要慢慢暖起来"。

30 分钟后:
  身体: 暖了
  心理: 开始有点无聊
  节奏: 可以稍微活跃一点

人的状态变了。共振结果也应该变。
```

## 三、原子的角色: 回应, 不是匹配

```
每个原子被赋予一个角色: "你能回应人的什么需要？"

weather:  "我能提供背景声音。你需要多安静？"
music:    "我能提供节奏和情绪。你现在需要什么速度？"
coffee:   "我一直是暖的。你需要品尝还是只需要温度？"
device:   "我能保护你不被打扰。你现在需要多沉浸？"
calendar: "我能告诉你还有多久。你现在需要被提醒吗？"

共振不是计算"weather × music = ?"。
共振是每个原子回答同一个问题:
  "这个人此刻需要什么？"
  然后互相参照: "你能给的, 和我能给的, 怎么一起更好？"
```

## 四、一个时刻的共振

```
你走进咖啡厅。坐下。打开降噪。窗外有雨。

共振不是一次性发生的。是一层层展开的:

第一层 (身体: 冷, 需要暖):
  coffee: "我在这里。一直是热的。不用等。"
  coffee × 窗外雨 → "外面冷, 这里面暖。喝一口。"
  → 不是"推荐一杯咖啡"。是"你已经有了, 它在等你"。

第二层 (心理: 需要安静, 不想被打扰):
  device: "降噪开着。外面的雨声还在, 但人的声音没了。"
  device × weather → "只剩下你和雨声。"
  → 不是"推荐降噪模式"。是"它已经在了, 你注意到了吗"。

第三层 (节奏: 慢, 不赶):
  calendar: "还有很久。什么都不用急。"
  music:    "我可以很慢。从最安静的开始。"
  calendar × music → "一首 8 分钟的曲子。刚好够你暖起来。"

每个人都在回应你的同一个状态。
不是原子之间匹配。
是原子们一起理解你。
```

## 五、实现

```
不做 atom × atom 的全局矩阵。

而是:
  1. 人的状态先被感知 (所有感知原子贡献)
  2. 人的状态 = 一个向量 (身体/心理/节奏/环境/时间)
  3. 每个原子算出自己对这个状态的回应值
  4. 回应值最高的几个原子自然聚在一起

原子不再互相算: "我和你能不能共振？"
原子各自算: "我能回应这个人的什么？"
回应相似的原子自然形成集群。
```

## 六、和之前的区别

```
之前 (原子中心):
  weather × music = 0.56
  weather × calendar = 2.0
  → 原子和原子共振

现在 (人中心):
  人的状态: 刚进来, 冷, 需要安静, 不赶时间

  weather:   "雨声 → 安静需求"     回应 0.9
  music:     "慢爵士 → 安静+慢需求"  回应 0.85
  coffee:    "暖 → 冷需求"          回应 0.95
  device:    "降噪 → 安静需求"       回应 0.9
  calendar:  "空闲 → 不需要赶"       回应 0.8

  coffee、weather、device 的回应最接近 → 自然聚在一起
  不是"weather和music共振"。
  是"这几个原子都在回应'你需要暖和安静'"。
```
