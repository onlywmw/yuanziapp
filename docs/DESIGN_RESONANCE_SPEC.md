## 四、共振计算

```python
def resonance(a, b):
    total = 0.0
    for cat_a, dims_a in a.field.items():
        for cat_b, dims_b in b.field.items():
            weight = 2.0 if cat_a == cat_b else 0.5
            for na, va in dims_a.items():
                for nb, vb in dims_b.items():
                    w = a.weights.get((na, nb), 0.5)
                    total += w * weight * va * vb
    return total
```

```
同类别 ×2, 跨类别 ×0.5
所有字段值统一归一化到 [0,1]
12 原子 × 5 维度 × 66 对 = < 1ms
```

## 五、权重学习

```
初始所有权重 = 0.5, 范围 [0.1, 0.9]

用户接受 → 参与维度权重 +0.05 × 时间衰减因子
用户忽略 → 参与维度权重 -0.03 × 时间衰减因子

时间衰减因子: 最近 7 天 ×1.0, 7-30 天 ×0.7, 30+ 天 ×0.3

判定:
  接受 = 用户点击/使用推荐 (3 分钟内)
  忽略 = 无操作超过 5 分钟
  拒绝 = 用户手动跳过/关闭

30 天后, 旧偏好随时间淡去, 新偏好自然浮现。
```
