# 人的两层 — 展示的 & 实际的

> **核心**: 人善于伪装。但伪装 ≠ 假的。伪装 = 想成为的样子。系统的角色不是拆穿, 是陪人走过去。

---

## 一、两层

```
展示层 (人想成为的):
  "我是早起的人"
  "我只听爵士"
  "我不需要社交"
  "我工作很专注"

实际层 (生活留下的痕迹):
  闹钟响 5 次, 11 点起床
  凌晨 2 点还在放嘻哈
  妹妹的电话 3 秒内接起
  下午 3 点开始刷手机

两层都是真的。
展示层: 向往的方向。
实际层: 此刻的位置。
```

## 二、不拆穿

```
不是:
  "你说你早起, 但你这周没有一天 8 点前起"
  → 这是嘲讽

是:
  系统默默记下展示层: "你想 8 点起"
  系统默默记下实际层: "这周最早是 10:30"
  然后:
    不提醒。不比较。不"帮你改进"。
    只是在 10:30 的时候, 比 11:00 的时候, 多一点温柔的音乐。
    像在说: "今天比昨天早了。好的。"
```

## 三、陪人走过去

```
展示层: "我只听爵士"
实际层: 凌晨 2 点放的是嘻哈

系统不评判:
  不把嘻哈藏起来
  也不在爵士列表里突然塞一首嘻哈然后说"我知道你其实喜欢这个"

  它只是:
    白天放爵士时, 多推一首有点嘻哈节奏的爵士
    像一座桥。不是一扇门。
    
  有一天:
    列表里有爵士, 也有那首凌晨放的嘻哈
    你没跳过
    展示的和实际的变成了同一个
```

## 四、当两层变成一个

```
不是系统"治好了"你的伪装。
是你不需要伪装了。

展示层: 早起
实际层: 8 点起
→ 一致了。不是因为系统逼你。
是因为每天 10:30 的音乐, 慢慢变成了 8:30 的。

展示层: "我不需要社交"
实际层: 妹妹的电话你在 3 秒内接
系统: 不评论。只是某天妹妹生日时, 提前 3 天轻轻提醒。
→ 你没关掉那个提醒。

不需要"变成更好的自己"。
只需要"现在的自己"和"想成为的自己"是同一个人。
```

## 五、实现

```
person-atom 的场有两部分:

  shown:     {早起:0.9, 爵士:0.9, 专注:0.8}
             ← 人自己说的, 或者通过选择表达出来的
             权重: 0.3 (系统不完全信)

  actual:    {早起:0.2, 嘻哈:0.6, 下午刷手机:0.7}
             ← 生活实际留下的痕迹
             权重: 0.7 (但系统也不完全信——人可能会变)

  两者之间的差距:
    差距大 → 系统保持沉默, 只是观察, 偶尔放一座桥
    差距缩小 → 系统轻轻靠近, 像在说"我注意到了"

  永远不做:
    比较、嘲讽、说教、擅自"帮人改进"
    "你应该..." ← 系统永远不会说这句话
```
