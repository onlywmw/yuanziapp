# 共振维度

> **问题**: 原子之间除了"雨强×慢节奏", 还能在哪些维度上共振?
> **分类**: 每个原子可以发出 5 类维度信号

---

## 物理维度 — 可测量的

```
强度:     rain=0.8, 音量=0.6, 光线=0.3
          → 大雨 × 大音量 × 暗光 → 共振

密度:     雨声密度 0.8, 爵士复杂度 0.9
          → 密集的雨 × 密集的音乐 → 共振

节奏:     雨滴速度 0.6, 音乐 BPM 0.5
          → 节奏相近 → 共振

变化率:   雨在变大 +0.3, 音乐在变快 +0.2
          → 都在加速 → 共振
```

## 时间维度 — 关于时间的

```
时长:     下了 2h, 空闲 2h
          → 刚好匹配 → 共振最强

周期性:   每天下午下雨, 每天下午听音乐
          → 不是今天才有的模式 → 共振

持续性:   连续下了 3 天, 连续听了 3 天爵士
          → 同一个模式持续中 → 共振

转折点:   雨刚开始(0-5min), 刚坐下(0-5min)
          → 都在"刚开始"的状态 → 共振
```

## 情绪维度 — 关于感受的

```
能量:     雨=0.3(低沉), 爵士=0.4(慵懒)
          → 能量水平相近 → 共振

开放度:   雨天=0.2(封闭), 降噪耳机=0.2(封闭)
          → 都在封闭状态 → 共振

熟悉度:   这个咖啡厅=0.9(常来), 这首Stan=0.9(常听)
          → 都是熟悉的东西 → 共振

新鲜度:   第一次来=0.1(陌生), 新专辑=0.1(陌生)
          → 都是新的体验 → 共振
```

## 状态维度 — 关于此刻

```
专注度:   降噪开=0.8(高度专注), 写代码=0.8
          → 都在专注 → 共振

移动性:   坐着=0.1, 慢爵士=0.1
          → 都是静止状态 → 共振

社交性:   一个人=0.0, 降噪=0.0(隔绝)
          → 都在独处 → 共振

舒适度:   靠窗=0.8, 熟悉音乐=0.8
          → 都很舒适 → 共振
```

## 关系维度 — 原子之间的

```
互补:     雨=冷(0.7), 咖啡=热(0.9)
          → 不是相同, 是互补 → 共振

因果:     雨→想要暖的, 咖啡=暖
          → 一个触发另一个 → 共振

对比:     外面吵=0.8, 降噪=0.9(安静)
          → 对立但相互成就 → 共振

协同:     雨声+爵士+咖啡
          → 三者在一起比两两更强 → 共振
```

## 维度组合示例

```
此刻的状态云包含所有这些维度:

weather: {
  物理: 雨强=0.8, 密度=0.9, 节奏=0.6
  时间: 持续=2h, 转折=0(不是刚开始)
  情绪: 能量=0.3, 开放度=0.2
}

music: {
  物理: 密度=0.9(爵士), 节奏=0.5(慢)
  时间: 持续=0(还没开始)
  情绪: 能量=0.4, 开放度=0.3
}

共振计算不再只是"雨强×节奏"。
而是所有维度交叉:

  物理×物理:   密度 0.9 × 密度 0.9 = 0.81   ← 强
  情绪×情绪:   能量 0.3 × 能量 0.4 = 0.12   ← 弱
  时间×时间:   持续 2h × 持续 0 = 0          ← 不共振
  
  但:
  物理.密度 × 情绪.开放度 → 0.18
  时间.持续 × 情绪.能量  → 0.6  ← 不明显的关联

几十个维度交叉, 最强的几个决定了最终输出。
不是人规定"雨和音乐在物理密度上共振"。
是系统自己算出: 这次是"持续时间×情绪能量"最强。
```
