# 图谱控制系统

> **定位**: 人怎么操控图谱 — 视角切换、参数微调、配色选择
> **合并**: GRAPH_MIXER_MODEL + GRAPH_PARAMETER_PANEL

---

## 一、混音台 — 视角切换

一根滑动条，两端是两个世界。

```
管道 ◄══════════●══════════► 作品

  管道端: 看系统血管 — 数据流动、运行记录、依赖关系
  作品端: 看人的创作 — 终端原子、风格标签、作者叙事
```

### 对图谱的影响

```
滑块位置决定哪些原子参与显示竞争:

  管道端:    只有管道原子 (mcp.*, connector.*, 基础原子) 参与
             → 混音台切换时, 不可见组的透明度降至 0%

  作品端:    只有终端原子 (有 soul 的注册原子) 参与
             → 中心节点: 终端原子放大, 管道节点缩小并淡出
  
  中间:      所有原子参与
```

### 预设按钮

```
[🔧 调试]   管道=1.0    紧凑/快速/标签配色
[⚡ 工作]   管道=0.7    适中/中速/标签配色
[🎨 浏览]   管道=0.3    舒展/慢速/分类配色
[💎 发现]   管道=0.0    舒展/慢速/风格配色
[🎛 默认]   管道=0.5    均衡/中速/标签配色
```

## 二、参数面板 — 微调

每个视觉维度一个推子：

```
节点大小      ○──────●────────○  1.2x
文字透明度    ░──────●────────█  70%
连线粗细      ──────●──────────  2dp
连线长度      ├──●──────────────┤  180dp
向心力        ○────●───────────○  0.5
节点排斥力    ○─────────●──────○  0.8
相连吸引力    ○──────●─────────○  0.6
布局速度      ○──────────●─────○  快
```

### 参数映射

| 推子 | 范围 | 影响 |
|------|------|------|
| 节点大小 | 0.5x ~ 2.0x | 全局节点缩放 |
| 文字透明度 | 0% ~ 100% | 标签可见度 |
| 连线粗细 | 0.5dp ~ 5dp | 连线描边宽度 |
| 连线长度 | 80dp ~ 300dp | 力导向 restLength |
| 向心力 | 0.0 ~ 1.0 | 节点向 CENTER 聚拢 |
| 排斥力 | 0.0 ~ 1.0 | 节点互相排斥 |
| 吸引力 | 0.0 ~ 1.0 | 相连节点吸引 |
| 布局速度 | 0.1 ~ 1.0 | 力导向收敛速度 |

### 混音台联动

```
混音台主控 → 所有推子跟随预设值
推子微调   → 当前预设变为"自定义"
点击预设   → 推子恢复预设值 (动画过渡 300ms)
```

## 三、配色方案 — 一键切换

```
[路径]   按 atom_id 命名空间着色    看来源地图
[标签]   按 classification.tags     看功能地图
[风格]   按 soul.style              看气质地图
[状态]   按 lifecycle.status        看状态地图
[属性]   按任意 JSON 路径           看自定义地图
```

### 配色规则

```
路径配色:
  system.*    → 灰色
  mcp.*       → 按域名分段着色
  connector.* → 绿色系

标签配色:
  Database    → amber
  Cloud       → clay_light
  Security    → rust
  AI          → rust_deep
  Document    → sage
  Sensor      → sage_light
  Connector   → green

风格配色:
  minimalist  → 白
  reliable    → 蓝
  creative    → 紫
  professional → 深蓝
```

### 切换动画

```
配色切换时:
  节点颜色渐变 (500ms)
  连线颜色同时变化
  颜色图例显示在右上角
```

## 四、存储

```
所有参数保存在本地 SharedPreferences (APK) / JSON 文件 (桌面):

  graph.mixer_position      = 0.5
  graph.node_scale          = 1.0
  graph.text_opacity        = 0.7
  graph.edge_thickness      = 2.0
  graph.edge_length         = 180
  graph.centripetal         = 0.5
  graph.repulsion           = 0.5
  graph.attraction          = 0.5
  graph.layout_speed        = 0.5
  graph.color_scheme        = "tag"

自定义预设最多 5 个。
启动时恢复上次参数。
```
