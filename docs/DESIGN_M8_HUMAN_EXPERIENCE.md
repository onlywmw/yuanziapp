# M8 人机体验层

> **灵感**: Obsidian 知识图谱 — 极客的浪漫、暗色的深邃、像在操控一个活的星系
> **原则**: 细节藏在深处, 表面极简, 人越深入越发现力量

---

## 一、打开 App 的感觉

```
不是"打开了一个工具"。
是"进入了自己的宇宙"。

  暗色背景从黑色淡入 (200ms)
  CENTER 节点先出现 — 像一颗恒星被点燃
  周围节点从黑暗中浮现 — 像星星一颗颗亮起
  连线以 CENTER 为起点向外生长 — 像光的轨迹
  最后一个节点稳定下来的时候
  整个图谱像星系一样, 缓慢旋转、微动、呼吸

没有"加载中"。没有进度条。只有光的出现。
```

## 二、节点: 不是 UI 组件, 是天体

### 光晕

```
每个节点有一个微小的外发光 — 像恒星的光晕

注册原子:
  正常: 光晕 2dp, 颜色=节点颜色, 透明度 20%
  hover: 光晕 6dp, 透明度 40%, 200ms 过渡
  select: 光晕 12dp, 透明度 60%, 向外脉冲一次

基础原子:
  无光晕, 哑光表面 — 它们是暗物质, 存在但不可交互

CENTER:
  常驻光晕 8dp, 纯白, 微呼吸 (亮度波动 ±10%, 周期 3s)
```

### 大小不是随机的

```
节点半径 = 8dp + 连接数 × 2dp

一个孤立的原子: 8dp 的小光点
连接 10 个邻居: 28dp 的中等节点
连接 50+ 个邻居: 100dp+ 的巨型节点 — 像一个恒星, 周围环绕着行星

大小是影响力。
不需要标注"重要节点" — 大的就是重要的。
```

### 颜色有意义

```
颜色不由设计师决定 — 由数据决定:

  路径配色:   同一命名空间 = 同一色系 — 看到组织的边界
  标签配色:   同一功能类别 = 同一色系 — 看到功能的地图
  soul配色:   同一风格 = 同一色系 — 看到气质的共鸣
  状态配色:   running=呼吸绿, offline=暗红, 未知=暗灰

颜色在讲数据自己的故事。
```

## 三、交互: 每一次触碰都有回应

### Hover

```
手指悬停在节点上方:

  该节点:          放大 1.15x, 光晕增强
  该节点的邻居:    微微亮起 (光晕 +10%)
  非邻居节点:      微微变暗 (透明度 -20%)
  与该节点相连的线: 变亮, 线宽 +1dp
  无关的线:        透明度降至 10%

像一个引力场 — 悬停的地方吸引了所有相关的东西。
```

### 双击聚焦

```
双击一个节点:

  1. 该节点平滑移动到屏幕中心 (弹簧动画, 400ms)
  2. 无关节点淡出 (alpha → 10%, 300ms)
  3. 该节点的直接邻居保持可见, 向外扩散
  4. 二级邻居更淡, 三级邻居几乎消失
  5. 标题栏出现面包屑: "全局 > mcp.postgres"
  6. 顶部出现深度滑块: [1] [2] [3] ← 显示几层邻居

这就是 Obsidian 的 Local Graph — 但它是原子的依赖图谱。
```

### 搜索

```
输入搜索词:

  1. 每输入一个字符, 匹配的节点亮起
  2. 不匹配的节点变暗 (不是消失 — 像星星被云遮住)
  3. 第一个匹配节点被一个 amber 色光环环绕
  4. 按回车: 平滑飞到第一个匹配节点
  5. 清除搜索: 所有节点恢复原状, 动画 500ms

搜索不是"筛选" — 是"聚光"。
```

## 四、力导向: 活的物理

### 不是算法, 是自然力

```
节点不应该机械地弹来弹去。
它们应该像在粘稠的液体里移动 — 有惯性, 有阻力, 有重量。

  新增节点: 从 CENTER 位置诞生, 被弹射到计算位置, 沿途留下微弱的光尾
  删除节点: 不是消失, 是爆炸成粒子, 邻居被推开
  拖拽节点: 像拉一根弹簧, 松手后惯性回弹
  力稳定后: 节点不静止, 微动 ±1-2dp — 像在呼吸
```

### 重力井

```
CENTER 节点像一个重力井。

所有节点被 CENTER 的引力牵引。
但牵引力随距离衰减:

  近处 (0-200dp): 强引力, 节点紧密环绕
  中距 (200-500dp): 正常引力
  远处 (500dp+): 极弱引力, 节点几乎自由漂浮

结果: 核心节点自然聚在中心, 外围节点散布在边缘。
像太阳系 — 近的行星快, 远的行星慢。
```

## 五、连线: 光的轨迹

### 不是线, 是光的路径

```
连线默认:
  极细 (1dp), 半透明 (alpha 15%)
  像两星之间微弱的引力线

Hover 节点时:
  相关连线变亮, 变粗 → 像激光被激活

数据流过时:
  红色/蓝色光粒子沿连线高速移动
  像血液在血管里流动
  粒子尾部有淡出的光尾

连线从不"突兀", 永远是背景。
只有在需要被看到的时候才亮起来。
```

### 连线不应该交叉

```
Bundle Rendering:

  多条同向连线走同一条路径 (像电路板走线)
  到目标节点附近再分叉

  这样:
    100 条连线 → 看起来像 20 条光路
    画面干净, 关系清晰
```

## 六、细节: 极客会注意到

### 帧率计数器

```
右上角, 极小字体, 暗灰色:

  60fps  → 不显示 (流畅时不打扰)
  <30fps → 显示, 暗红色 "29fps"
  
  极客爱看帧率。
```

### 节点信息提示

```
长按节点, 不要弹窗 — 在节点旁边浮现一行小字:

  "mcp.postgres · ⭐4.7 · 作者张三 · 7个功能"

  松手即消失。
  不给弹窗, 不给按钮 — 只有信息。
```

### 双击空白

```
双击空白处:

  不是"重置视角" — 是"回到 CENTER"。
  
  CENTER 节点平滑移到屏幕中央。
  所有节点跟随, 像被吸回去。
  弹簧动画 400ms, 带有轻微的过冲回弹。
```

### 缩放指示器

```
缩放时不显示数字 "1.5x"。
在右下角画一条极小的线:

  |-----------●------------|

  点在最左 = 0.3x, 在最右 = 3.0x
  无声, 不打扰, 但精准。
```

## 七、声音 (可选, 极客开关)

```
默认关闭。在设置里可以打开。

  节点出现:   微弱的"啵" (类似 macOS 的 Pop 音, 但更轻)
  hover 节点:  极微弱的嗡声 (类似磁力)
  双击聚焦:   低沉的"嗡—" (像一个大的物体在共振)
  数据流过:   微弱的电流声 (高频沙沙声)

不是"音效", 是"声音纹理"。
极低音量, 混在环境音里。
像 Dune 电影的原声 — 不是为了引起注意, 是为了创造沉浸。
```

## 八、调音台: 不挡画面的控制

```
左下角一个小齿轮图标。

点击 → 不是弹窗, 是从底部滑出一个半透明面板:

  ┌──────────────────────────────────────────┐
  │  管道 ◄══════════●══════════► 作品      │
  │                                          │
  │  [🔧] [⚡] [🎨] [💎] [➕]               │  ← 预设
  │                                          │
  │  颜色 [路径|标签|风格|状态]              │
  │  更多参数 ▸                              │  ← 展开高级面板
  └──────────────────────────────────────────┘

面板半透明, 高斯模糊背景。
不覆盖图谱, 图谱在面板后面继续呼吸。
```

## 九、极客的浪漫

```
为什么 Obsidian 的图谱让人上瘾:

  不是因为它有用 — 是因为它美。
  不是因为功能多 — 是因为每个细节都被照顾到了。
  不是因为它强大 — 是因为它让你感觉自己像在操控一个宇宙。

Yuanzi 的图谱应该给同样的感觉:

  你的原子像星星。
  原子之间的连接像引力。
  数据流过时像血液。
  而你 — 站在宇宙之外, 用手指轻轻拨动它。

这不是一个工具。这是你的宇宙。
```
