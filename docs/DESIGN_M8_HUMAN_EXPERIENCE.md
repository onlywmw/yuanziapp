# M8 人机体验层

> **架构**: 地基 (Graph SDK 钩子系统) + 模板层 (审美注入)
> **原则**: 地基零审美, 模板全审美。Graph SDK 可单独复用, M8 模板是"穿在 SDK 外面的衣服"。

## 二、地基暴露的钩子

Graph SDK 不内置任何审美。它只暴露钩子——让模板层注入审美。

```
// SDK 暴露的钩子 (纯函数接口)

onNodeRender(node, canvas, state):     // 节点绘制前 — 模板决定画成恒星还是方块
onEdgeRender(edge, canvas, state):     // 连线绘制前 — 模板决定画成引力线还是实线
onBackground(canvas):                   // 背景 — 模板决定暗色还是亮色
onNodeAppear(node):                     // 节点出现 — 模板决定扩散还是弹入
onNodeDisappear(node):                  // 节点消失 — 模板决定爆炸还是淡出
onHover(node, neighbors):               // 悬停 — 模板决定邻居亮起还是放大
onFocus(node, depth):                   // 局部图谱 — 模板决定聚光还是框选
onSearch(matches):                      // 搜索 — 模板决定高亮还是遮罩
onDragStart(node):                      // 拖拽开始
onDragEnd(node):                        // 拖拽结束 — 模板决定弹簧回弹还是惯性
onDataFlow(edge, progress):             // 数据流 — 模板决定粒子还是脉冲
onLayoutTick(forces):                   // 力导向每帧 — 模板可注入额外力

// 模板注册:
graph.registerTemplate("obsidian", obsidianTheme)
graph.applyTemplate("obsidian")
```

## 三、模板: Obsidian 星系

这是 M8 的第一个模板。灵感来自 Obsidian 知识图谱。

### 3.1 节点 → 恒星

```
onNodeRender:
  基础: 圆形, 底色=按配色方案, 无边框
  光晕: 节点外 2dp 高斯模糊光晕, 颜色同节点色, alpha 20%
  大小: 半径 = 8dp + 连接数 × 2dp (连接越多越亮越大)
  CENTER: 半径 24dp, 纯白, 呼吸光晕 (亮度波动 ±10%, 周期 3s)
```

### 3.2 连线 → 引力线

```
onEdgeRender:
  基础: 1dp 细线, 颜色=节点色, alpha 15%
  悬停时: 粗 3dp, alpha 80%, 像激光被激活
  Bundle: 多条同向线走同一路径, 到目标附近分叉

onDataFlow:
  红色光粒子 (动脉) / 蓝色光粒子 (静脉)
  沿连线平滑移动, 尾部带淡出光尾
```

### 3.3 背景 → 深空

```
onBackground:
  纯黑 #0D0D0D (比 Obsidian 的 #1A1A1A 更深)
  缩放 > 0.8x 时: 极淡的六角网格 #151515, 像星图坐标
  缩放 < 0.3x 时: 节点缩小为光点, 像深空照片里的星星
```

### 3.4 交互动画

```
onHover:
  目标节点放大 1.15x, 光晕扩大
  邻居节点微亮 (alpha +10%)
  非邻居节点微暗 (alpha -20%)
  相关连线变亮变粗
  无关连线 alpha → 10%

onFocus (双击):
  目标节点弹簧动画移到屏幕中心 (400ms)
  一级邻居可见, 二级半透明, 三级以上不可见
  顶部面包屑: "全局 > 节点名"
  深度滑块: [1] [2] [3]

onSearch:
  匹配节点高亮 (amber 光环环绕)
  不匹配节点变暗 (alpha 降至 20%, 不是消失)

onDragEnd:
  弹簧回弹, 带轻微过冲
```

### 3.5 粒子效果

```
onNodeAppear:
  节点从 CENTER 位置 scale 0 → 1, 沿贝塞尔曲线飞向目标位置
  光尾: 路径上残留淡出的光点 (3-5 个)

onNodeDisappear:
  节点爆炸为 8-12 个微粒子
  粒子向外飞散, 淡出, 持续 400ms
  邻居被推开 ±10dp
```

### 3.6 极客细节

```
帧率: 右上角, 极小字体, 暗灰。≥55fps 不显示, <30fps 暗红显示 "29"
缩放指示器: 右下角, 一条细线 + 点, 无声
声音: 默认关闭。开启后有关节点出现(啵)/悬停(嗡)/聚焦(嗡—)/数据流(沙沙)
```

## 四、模板: 极简 (未来)

```
比 Obsidian 更素:
  无光晕
  无粒子
  节点 = 纯几何形状
  连线 = 1px 实线
  背景 = 纯白或纯黑
  文字 = 系统字体
```

## 五、模板: 赛博朋克 (未来)

```
高对比、霓虹、数据感:
  背景 = 深蓝黑
  节点 = 霓虹边框, 无填充
  连线 = 霓虹色虚线
  数据流 = 高速脉冲
  文字 = 等宽字体
```

## 六、模板是用来换的

```
未来模板市场:

  用户可以选择自己喜欢的模板。
  创作者可以上传模板。
  模板 = 一个 JSON 配置文件 + 可选的自定义渲染函数。

  切换模板时:
    不是"刷新页面" — 是跨渐变动画
    节点形状、颜色、连线风格平滑过渡
    像换了一套皮肤
```
